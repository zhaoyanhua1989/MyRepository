发送消息 1、Hanlder 创建的过程中，在主线程中，可以不定义Looper，默认为主线程的Looper。工作线程中需要定义Looper。 处理机制： 1、Handler h = new Handler(){重写handleMessage}; h.handleMessage(msg); 2、final ProgressDialog dialog = new ProgressDialog(this); Handler h = new Handler(); h.post(new Runnable() { @Override public void run() { num++; if (num <= 10) { dialog.setProgress(num); h.postDelayed(this, 1000); } else { dialog.cancel(); num = 0; } } }); 3、Message.obtain(Handler h, int what, int arg1, int arg2, Object obj); 2、循环发消息的实现： 1、线程、Hanlder 线程runnable中套死循环while(true)， 循环中，不能用postDelayed实现，不然delayed的时间一到，会收到n条挤在一起的消息。 循环中，应该用Thread.sleep()来实现消息的暂停。然后借助Hanlder来传递消息和处理消息。 2、Hanlder 用Hanlder发送空消息sendEmptyMessageDelayed来实现，在handleMessage中处理msg.what， 收到对应消息，再发送同样msg.what的空消息sendEmptyMessageDelayed。
