单元测试 Android的单元测试(JUnit方式)： 1. 在工程中创建测试类，继承TestCase，随便定义一个测试方法，实现需要测试的代码； 2. 测试类Run As JUnit Test，弹出错误提示，在Run按钮下选择Run Configurations； 3. 弹出配置页面后，选择JUnit --> Test(2) --> Classpath --> Bootstrap Entries --> Android... --> remove， --> Advanced --> Add Library --> 分别添加JRE System Library 和 JUnit3，然后保存 4. 测试类，Run As JUnit Test，就可以了。
